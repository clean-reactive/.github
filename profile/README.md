# Hello, I am Clean Reactive Architecture :wave:

Clean Reactive Architecture is a practice for building reactive client
applications - applications powered by Jetpack Compose, SwiftUI, React,
Angular, Vue, or any framework based on the observer pattern.

The practice extends and rearranges existing architectural concepts and
methodologies into a consistent, self-sufficient, and composable application
architecture and development methodology.
